# Classmate Birthday Manager

A Desktop Application to manage classmate birthdays using JavaFX and MySQL.

## Features
- **Add, Update, Delete**: Manage student birthday records easily.
- **Search**: Search classmates by name.
- **Bangla UI**: User interface labels and buttons are in Bangla.
- **Notifications**: Automatically detects and alerts if any classmate has a birthday today.
- **Database**: Stores all data persistently using MySQL.

## Requirements
- Java Development Kit (JDK) 8 or higher
- MySQL Database
- `mysql-connector-java` library

## Setup
1. Import the project into your IDE (IntelliJ/Eclipse).
2. Add the JDBC driver jar file to the library.
3. Run the SQL script to create the database.
4. Update the DB credentials in `Main.java` if necessary.
5. Run `Main.java`.
